Step-by-step installation (manual CRD + chart):

1. Install CRD first:
   kubectl apply -f chart/harness-gitops-agent-controller/crds/harnessgitopsagents.infrastructure.kandylis.co.uk.yaml

2. Install/upgrade chart without CRD management:
   helm upgrade --install {{ .Release.Name }} {{ .Chart.Name }} \
     --namespace {{ .Release.Namespace }} \
     --create-namespace \
     --skip-crds

3. Verify resources:
   kubectl get crd harnessgitopsagents.infrastructure.kandylis.co.uk
   kubectl get deploy -n {{ .Release.Namespace }} {{ include "harness-gitops-agent-controller.fullname" . }}

4. Follow controller logs:
   kubectl logs -n {{ .Release.Namespace }} deploy/{{ include "harness-gitops-agent-controller.fullname" . }} -f
